About Sauti

A mobile app that uses advanced text-to-speech and speech-to-text technology to bridge the communication gap for those with hearing and speech impairments.

Our Values

Inclusivity

We believe in breaking down barriers and fostering an environment where everyone, regardless of their abilities, can communicate and connect seamlessly.

Empowerment

Our goal is to provide people with the tools and technology they need to express themselves confidently, ensuring that they have a voice in every conversation.

Accessibility

We are committed to ensuring that Sauti is available to all, especially those in underserved or developing regions, making it a practical, affordable solution for everyone in need.

Flutter

Flutter is an open-source UI toolkit by Google that enables developers to build natively compiled applications for mobile, web, and desktop from a single codebase.

Sauti app was built using Flutter to create a responsive and visually appealing user interface, ensuring consistent performance across different platforms like Android and iOS.

Mobile App development

Mobile app development involves creating software applications that run on mobile devices, optimized for user interaction and functionality.

The Sauti app utilized mobile app development principles to integrate Speech-to-Text (STT) and Text-to-Speech (TTS) features, making it accessible and functional for users on the go.

Speech Recognition and Synthesis API's

APIs like speech_to_text and flutter_tts are tools that allow developers to implement voice recognition and text-to-speech functionalities in their applications.

These APIs were used to enable voice input for converting speech into text (STT) and to provide audio output by converting text into speech (TTS) with adjustable settings like volume, pitch, and rate.
